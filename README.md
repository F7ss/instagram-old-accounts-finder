# instag# Instagram Old Accounts Finder

أداة للبحث عن حسابات إنستقرام القديمة (2010-2016)

## المميزات
- البحث عن حسابات قديمة
- استخراج الإيميلات من البيو
- تصدير النتائج لملف نصي أو إرسالها عبر تليجرام

## المتطلبات
- Python 3.8+
- See requirements.txt
